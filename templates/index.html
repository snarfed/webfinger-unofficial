WebFinger for {{ domain }}

This is an unofficial WebFinger server for {{ domain }}. It serves these endpoints:

  • https://{{ host }}/.well-known/host-meta
  • https://{{ host }}/.well-known/webfinger?resource={uri}
  • https://{{ host }}/user?uri={uri}

The user URI is acct:username@{{ domain }}, where username is a username or user id. See the WebFinger docs for usage details.

Source: github.com/snarfed/webfinger-unofficial
